# Codofolio

A platform for connecting developers, experts, and freelancers.

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/teambugbusters00/codofolio.git
   cd codofolio
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

The application will be available at `http://localhost:8080`.

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router
- Appwrite
